# msg-launcher

---

## 🇷🇺 Описание проекта
**MSG Launcher** — это современный кастомный лаунчер для Minecraft Bedrock Edition на Android, написанный с нуля на Kotlin. Проект ориентирован на предоставление удобного интерфейса, встроенного магазина дополнений и продвинутых инструментов отладки для игроков и разработчиков.

### ✨ Ключевые особенности:
* **Полная интеграция с Firebase:** Быстрая синхронизация данных, авторизация и стабильный бэкенд.
* **Встроенный Marketplace:** Удобный просмотр, скачивание и автоматическая установка модов, текстур и дополнений на лету.
* **Продвинутый Logcat оверлей:** Мощные инструменты логирования и встроенная отладка ошибок прямо в приложении.
* **Плавный интерфейс:** Поддержка горизонтального скролла в меню выбора версий и кастомные анимации.
* **Мультиязычность:** Встроенная поддержка нескольких языков.

---

## 🇬🇧 Project Description
**MSG Launcher** is a modern custom launcher for Minecraft Bedrock Edition on Android, built from scratch using Kotlin. The project is designed to provide a smooth user experience, an integrated asset marketplace, and advanced debugging tools for both players and developers.

### ✨ Key Features:
* **Firebase Integration:** Reliable backend setup, authorization, and fast data sync.
* **Built-in Marketplace:** Easily browse, download, and auto-install mods, textures, and add-ons on the fly.
* **Advanced Logcat Overlay:** Powerful logging and runtime debugging tools built right into the app.
* **Fluid UI:** Horizontal scrolling in the version selector and marketplace, paired with custom animations.
* **Multi-language Support:** Native localization support for multiple languages.
